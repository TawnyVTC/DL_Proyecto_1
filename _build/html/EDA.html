Se cre√≥ la variable binaria ‚Äòdefault‚Äô en base a la variable original ‚Äòloan_status‚Äô. Se tiene ‚ÄòFully Paid‚Äô = 0 y ‚ÄòCharge Off‚Äô/‚ÄôDefault‚Äô = 1. Luego se elimina aquellos registros en los que ‚Äòdefault‚Äô seria nula.
